## About

Backend-focused software engineer with 3+ years of experience building and shipping full-stack web applications. I specialize in TypeScript, Next.js, Node.js, and PostgreSQL — with a growing focus on backend systems architecture, AI integration.

Currently pursuing a B.E. in Computer Software Engineering at Mehran University of Engineering and Technology, Hyderabad, while building real products used by real people.

- Currently building: [NexLink](https://github.com/vishaldewani206) — an open-source Linktree alternative in TypeScript
- Learning: PostgreSQL internals · DSA with Python · Three.js · RAG pipelines
- Writing about what I build at [vishaldewani.vercel.app/blog](https://vishaldewani.vercel.app/blog)
